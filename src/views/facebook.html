H.W.C is an autonomous high-rise window washer, it provide a low cost and efficient solution for cleaning the high-rise glass surface to instead of manual cleaning. In this page i will introduce the design process of the mobile interface.

To start with, we investigated the operational process of the target user when they are working. And through the background research, we collated and referenced the mobile interface of the existing products. To ensure that the app stays relevant to the user and achieves the best user experience, We continued to improve our interactive during the production process.

We designed the main workflow for the app based on the operation process of the high-rise window washer.Scan to determine the number of building floors and the number of windows, automated cleaning, reminders after cleaning, intimate history, and convenient customer service.

THE HOME SCREEN

The main screen is designed to provide direct access to all content in the app. Users can select the cleaning mode, find the history, and view the summary. users can know the weather by looking at the weather bar at the top, and set the real-time information of the drone on the right side of the weather bar. In addition, the navigation bar at the bottom can enter the customer service consultation, start page, history page, and the user can return to the main screen at any time through the navigation bar.

A SEAMLESS EXPERIENCE

To ease the experience the site was designed to seamlessly take the user to different stories on the homepage. Text, icons and panels moving smoothly to not interrupt the user flow.
